# Lotus UI Fix Prompt

Make the UI closer to the Figma Make reference.

Fix these specifics:

1. Color system
Use warm ivory, cream, peach, soft beige, and subtle gold accents. Avoid cold gray and harsh white.

2. Header placement
Keep the top bar clean and aligned like Figma Make:
- Lotus logo left
- Version selector
- Route/path pill
- Device switcher centered
- Assistant/router tools right
- Publish/Deploy button far right

3. Chat composer
Remove bottom clutter. Only show:
- plus button
- router/model switcher
- input
- send button

4. Plus menu
Clicking plus opens real app components:
- Upload File
- Upload Image
- Add Connector
- Add Skill
- Add Agent
- Add Function
- Import GitHub Repo
- API Keys
- Environment Variables
- Device Capabilities

5. Router switcher
Put router switcher directly inside the composer, not hidden in a top nav.

6. Connectors
Connectors should be a settings-style hub. No drag-and-drop.

7. Preview
Keep the phone preview movable/resizable if possible, but do not make the entire UI chaotic.

8. Code
Code tab should support file tree, copy, download, export ZIP.

9. Deployment
Rename Publish to Deploy where appropriate. Keep View App separate.
